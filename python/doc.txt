[Add Two Numbers]
Ввод: два списка представляющие два положительных числа, цифры чисел находятся в обратном порядке.
Вывод: Сложить два числа и вернуть сумму в виде списка
Пример:
- Ввод: l1 = [2,4,3], l2 = [5,6,4]
- Вывод: [7,0,8]
- Операция: 342 + 465 = 807

[Reverse Integer]
Ввод: целое число (положительное или отрицательное)
Вывод: число наоборот
Пример 1:
- Ввод: x = 123
- Вывод: 321
Пример 2:
- Ввод: х = -123
- Вывод х = -321

[Palindrome Number]
Ввод: Целое число (положительное или отрицательное)
Вывод: true/false (палиндром/не палиндром)
Пример 1:
- Ввод: x = 121
- Вывод: true
Пример 2:
- Ввод: x = -121
- Вывод: false
